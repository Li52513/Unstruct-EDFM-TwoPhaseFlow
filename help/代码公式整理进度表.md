# 代码反推公式逐文件清单

> 组织原则：始终保持“网格划分/几何定义 -> 裂缝离散与求交 -> 拓扑与全局编号 -> 场与物性 -> 本构与传输系数 -> 边界与装配 -> 全隐式求解 -> 后处理”的主线不变。  
> 编排原则：按论文章节挂接文件清单；同一文件可在不同章节重复出现，但每次只提炼该章节所需公式。  
> 使用原则：上传代码时尽量按“小节”为单位，不要跨多个小节混传。
